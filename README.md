## 主なプロジェクト

### お絵描きシューティングゲーム
- Electron + TypeScript + Python + 機械学習を用いて、手描きのフルーツを弾として使える体験型シューティングゲームを開発
- 東京大学 5 月祭で精密工学科の企画として展示し、約 200 名が体験、100 名以上がクリア
- [Fruit_shooting](https://github.com/kanta341/Fruit_shooting)

### 就活AI
- Next.js + Supabase + NextAuth
- AIを用いたアイディア発想とSNSでの共有を組み合わせたWebアプリ
- [idea-SNS](https://github.com/kanta341/idea-SNS)

### アイディア共有SNS
- Next.js + Supabase + NextAuth
- AIを用いたアイディア発想とSNSでの共有を組み合わせたWebアプリ
- [idea-SNS](https://github.com/kanta341/idea-SNS)

### マルチエージェントシミュレーション（チーム単位での研究プロジェクト）
- Pythonを用いて大学構内の食堂混雑緩和をテーマにシミュレーションを実施
- データ収集・基礎モデル構築を担当

### 迷路の難易度判定AIモデル構築
- TensorFlowを用いて迷路の難易度を推定するAIモデルを構築
- データ収集からモデル設計まで一貫して実施
- 強化学習やルールベース前処理を試行
